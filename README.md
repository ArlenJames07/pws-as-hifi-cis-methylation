## Overview
PacBio HiFi workflows for haplotype- and molecule-resolved methylation, CNV, SV and phasing analysis of the PWS/AS 15q11–q13 imprinted domain.


This repository contains the reproducible analysis scripts, configuration files, and non-identifiable processed summary tables supporting the manuscript “Reciprocal imprinting disorder deletions reveal coordinated cis-methylation architecture of the human 15q11–q13 domain.” The workflow uses PacBio HiFi long-read sequencing aligned to T2T-CHM13v2.0 to integrate structural variation, copy-number profiling, haplotype phasing, native CpG methylation, boundary mapping, and single-molecule methylation analyses across the Prader–Willi/Angelman syndrome imprinted domain.

The repository is intended to support transparent reproduction of the main analyses and figures, including imprinting-centre methylation validation, reciprocal parental methylation reconstruction, SNHG14/ICR-proximal boundary detection, SNORD116 molecule-level methylation analysis, and breakpoint-flanking methylation assessment. Raw human genomic data are not included because of participant privacy and consent restrictions.
